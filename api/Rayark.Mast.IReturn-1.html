Interface IReturn<T>

Provides interface for an iterator block to return a value or report an error.

Namespace: Rayark.Mast
Assembly: cs.temp.dll.dll
Syntax
public interface IReturn<T>
Type Parameters
Name Description
T

The type of the returned value.
Remarks

Typically IReturn<T> is used as an argument of an iterator block, which is meant to be executed through BlockMonad<T>.

Examples

Sleep 1 second and return a value of the float input.

IEnumerator Main(){
    var m = new BlockMonad<float>( r => DelayFloat(3.0f, r));
    yield return m.Do();
    Assert.That( m.Result == 3.0f );
}

static IEnumerator DelayFloat(float input, IReturn<float> ret){
    yield return Coroutine.Sleep(1.0f);
    ret.Accept(input);
}
