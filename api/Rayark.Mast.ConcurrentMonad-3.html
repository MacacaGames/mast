Class ConcurrentMonad<T1, T2, T3>

Executes three monads with different or the same type of return value concurrently. When any monad finished with error, the CorrurentMonad will stop immediately.

Remarks

When any monad finished with error, the ConcurrentMonad<T1, T2, T3> will stop immediately. This class is rarely used directly. Use the extension method WhenAll<T1, T2, T3>(IMonad<T1>, IMonad<T2>, IMonad<T3>) or LINQ syntax instead if it is possible.
